sobre o projeto

Espectrograma como interface pedagógica

Este aplicativo foi desenvolvido como ferramenta didática para a disciplina Criação Musical com Novos Suportes Tecnológicos (UNICAMP/NICS), permitindo que estudantes explorem visualmente a relação entre representação espectral e síntese sonora.

A proposta central é tornar o espectrograma — normalmente uma representação de análise — em uma superfície de criação: o aluno desenha, e o sistema sintetiza. Isso inverte o fluxo analítico tradicional e abre espaço para experimentação composicional intuitiva.

✏️

Desenho livre

Pincéis coloridos mapeados a frequências. Intensidade pelo brilho do pixel.

📂

Importação de imagem

Qualquer .jpg ou .png vira espectrograma tocável. Rostos, texturas, pinturas.

◈

Visualização heatmap

Paletas inferno, turbo, viridis e ciano — como espectrogramas científicos reais.

🔊

Síntese aditiva

Web Audio API. Osciladores por coluna com envelope ADSR e 4 tipos de onda.

como usar

Guia rápido de uso

1. Desenhar e ouvir

Use os pincéis coloridos para desenhar no canvas. Cada posição vertical corresponde a uma frequência (escala logarítmica). Posição horizontal = tempo. Clique em ▶ reproduzir para ouvir o resultado.

2. Linhas harmônicas

Selecione o modo ╱ linha e arraste para criar harmônicos paralelos, glissandos ou melodias. Linhas diagonais produzem glissandos; horizontais produzem notas sustentadas.

3. Importar imagem

Clique em 📂 importar .jpg, escolha qualquer imagem. Em seguida clique em ◈ ver espectrograma para visualizar com paleta científica e reproduzir.

4. Ajustar parâmetros

No painel lateral, controle duração total, faixa de frequências (grave/agudo), tipo de onda (senoidal, triangular, quadrada, dente-de-serra) e tempo de ataque de cada oscilador.

5. Paletas de cor

Na aba ◈ espectrograma, escolha entre as paletas inferno, turbo, viridis ou ciano para diferentes representações visuais da mesma informação espectral.
